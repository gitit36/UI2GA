# UI2GA - AI 기반 GA4 태깅 자동화 아키텍트

UI2GA는 UI 스크린샷을 분석하여 전문가 수준의 **GA4(Google Analytics 4) 데이터 태깅 기획안**을 자동으로 생성하는 혁신적인 AI 도구입니다. Google Gemini Vision AI를 활용하여 화면 내 모든 인터랙션 요소를 탐지하고, 국내 엔터프라이즈급 분석 표준에 맞춘 명세서를 즉시 제공합니다.

---

## ✨ 핵심 가치

데이터 분석 설계는 기획자와 개발자 사이의 많은 커뮤니케이션 비용을 발생시킵니다. UI2GA는 이 과정을 자동화하여 다음과 같은 가치를 제공합니다.

1.  **휴먼 에러 방지**: 수동으로 작성하던 태깅 리스트의 누락을 방지하고 명칭 규칙(Naming Convention)을 통일합니다.
2.  **생산성 극대화**: 스크린샷 업로드 한 번으로 수십 개의 이벤트 명세서와 주석(Annotation) 이미지를 생성합니다.
3.  **커뮤니케이션 일치**: 개발자에게 전달할 시각적 가이드(번호가 매겨진 주석 이미지)를 함께 생성하여 오해의 소지를 없앱니다.

---

## 🛠 주요 기능

### 1. 정밀 UI 요소 탐지 (AI Vision)
*   **자동 인식**: 버튼, 입력창, 탭, 토글, 리스트 아이템 등 모든 클릭 가능한 요소를 AI가 스스로 판단하여 추출합니다.
*   **부분 영역 분석**: 화면 전체가 아닌 특정 영역만 드래그하여 해당 부분에 대한 집중 태깅 설계가 가능합니다.

### 2. 고도의 태깅 로직 (Analytics Intelligence)
*   **Event Category**: 화면의 성격에 맞는 고정 식별자 자동 생성 (예: `통합검색_자연어검색`).
*   **Event Action**: 행위의 강도에 따른 접두사(Prefix) 자동 적용
    *   `tap_`: 가벼운 탐색 또는 탭 이동
    *   `click_`: 확인, 제출, 결제 등 중요한 동작
    *   `select_`: 리스트나 옵션 선택
    *   `view_`: 특정 영역이나 모듈의 노출
*   **Event Label**: 변수 처리가 필요한 항목(검색어, 뉴스 제목 등)을 `{}` 중괄호로 구분하여 개발 가이드화.

### 3. 강력한 인터랙티브 워크스페이스
*   **실시간 편집**: AI가 생성한 결과물을 테이블에서 즉시 수정 가능하며, 수정 시 JSON 데이터도 실시간 동기화됩니다.
*   **주석 레이어**: 이미지 위에 번호가 매겨진 빨간색 박스가 표시되어 어떤 UI 요소가 어떤 이벤트인지 한눈에 확인 가능합니다.
*   **이미지 제어**: Space + 드래그를 통한 이동 및 휠을 이용한 정밀한 확대/축소(Zoom)를 지원합니다.

### 4. Figma(피그마) 양방향 연동
*   **가져오기**: 피그마 프레임을 선택해 UI2GA로 바로 전송하여 분석할 수 있습니다.
*   **내보내기**: 분석이 완료된 태깅 테이블과 주석 이미지를 SVG 형태로 피그마 캔버스에 직접 삽입할 수 있습니다. (피그마 플러그인 모드)

## 📂 데이터 내보내기 형식

작업 결과물은 다양한 협업 도구에 맞게 내보낼 수 있습니다.
*   **Markdown**: 노션(Notion)이나 컨플루언스(Confluence)에 복사-붙여넣기 최적화.
*   **CSV / XLSX**: 엑셀 기반의 데이터 명세서 관리용.
*   **JSON**: 개발자가 즉시 참고할 수 있는 데이터 구조.
*   **SVG (Figma)**: 디자인 도구 내 가이드 레이어 삽입용.
*   **Annotated Image**: 주석이 포함된 고해상도 PNG 이미지.

## ⌨️ 단축키 및 팁

*   **영역 선택**: 이미지 위를 드래그하여 분석할 영역을 지정하세요.
*   **화면 이동**: `Space` 키를 누른 상태에서 마우스를 드래그하세요.
*   **확대/축소**: 마우스 휠을 사용하세요.
*   **셀 수정**: 테이블의 셀을 더블 클릭하면 내용을 바로 수정할 수 있습니다.
*   **커스텀 규칙**: 우측 'Rules & Context' 영역에 "모든 버튼에 tap_ 접두사 사용"과 같은 지침을 입력하면 AI가 이를 반영합니다.
